# Ferramentas de Acessibilidade

Para auxiliar e facilitar o desenvolvimento de soluções acessíveis, foram elencadas ferramentas de acessibilidade abaixo. 

---

##  Validação e Testes Automáticos

- [**Accessibility Insights (Microsoft)**](https://accessibilityinsights.io/) — Ferramenta gratuita da Microsoft que realiza auditorias automáticas e guiadas com base nas WCAG.  
- [**Tenon.io**](https://tenon.io/) — Validador online que analisa o código das páginas e aponta erros de acessibilidade.  
- [**SiteImprove Accessibility Checker**](https://siteimprove.com/en/accessibility/accessibility-checker/) — Extensão para Chrome que gera relatórios de acessibilidade em tempo real.  
- [**Pa11y**](https://pa11y.org/) — Ferramenta de linha de comando que testa acessibilidade e gera relatórios em HTML ou JSON.  
- [**SortSite**](https://www.powermapper.com/products/sortsite/) — Verifica erros de acessibilidade, SEO e usabilidade em todo o site.  
- [**AXE - Accessibility Engine**](https://github.com/dequelabs/axe-core) — Motor de testes automatizados para interfaces web acessíveis.  
- [**Lighthouse (Google)**](https://developer.chrome.com/docs/lighthouse) — Ferramenta do Google para auditar desempenho, SEO e acessibilidade.  
- [**ACHECKS**](https://www.achecks.org/) — Conjunto de ferramentas online para verificar conformidade com padrões de acessibilidade.  
- [**Wave - Web Accessibility Evaluation Tool**](https://wave.webaim.org/) — Valida automaticamente e exibe alertas visuais sobre problemas de acessibilidade.  
- [**Framework de Testes de Acessibilidade (Google)**](https://github.com/google/Accessibility-Test-Framework-for-Android) — Framework para testes de acessibilidade em aplicativos Android.  
- [**DynoVisual Sitemap Generator**](https://dynomapper.com/) — Gera mapas visuais e faz auditoria de acessibilidade das páginas.  

---

##  Cores, Contraste e Design Inclusivo

- [**WebAIM Contrast Checker**](https://webaim.org/resources/contrastchecker/) — Verifica o contraste entre cores conforme as diretrizes WCAG.  
- [**Color Oracle**](https://colororacle.org/) — Simula tipos de daltonismo para verificar legibilidade das cores.  
- [**Stark**](https://www.getstark.co/) — Plugin para Figma, Adobe XD e Sketch que ajuda a testar contraste e acessibilidade visual.  
- [**Contrast Ratio (Lea Verou)**](https://contrast-ratio.com/) — Teste simples e rápido da relação de contraste entre cores.  
- [**Tanaguru Contrast Finder**](https://contrast-finder.tanaguru.com/) — Sugere automaticamente combinações de cores acessíveis.  

---

##  Leitores de Tela e Simuladores

- [**JAWS**](https://www.tecassistiva.com.br/catalogo/jaws/) — Leitor de telas comercial para Windows, amplamente utilizado.  
- [**NVDA (NonVisual Desktop Access)**](https://www.nvaccess.org/) — Leitor de telas gratuito e open source para Windows.  
- [**VoiceOver (Apple)**](https://support.apple.com/voiceover) — Leitor de tela integrado ao macOS e iOS.  
- [**TalkBack (Android)**](https://support.google.com/accessibility/android/answer/6283677?hl=pt-BR) — Leitor de tela padrão dos dispositivos Android.  
- [**ChromeVox**](https://chrome.google.com/webstore/detail/chromevox-classic-extensi/kgejglhpjiefppelpmljglcjbhoiplfn) — Leitor de telas do navegador Google Chrome.

##  Análise Manual e Educação

- [**WCAG-EM Report Tool (W3C)**](https://www.w3.org/WAI/eval/report-tool/) — Ferramenta oficial do W3C para criar relatórios de conformidade WCAG.  
- [**Inclusive Design Principles**](https://inclusivedesignprinciples.org/) — Guia com princípios de design inclusivo aplicáveis a qualquer projeto digital.  
- [**A11y Project Checklist**](https://www.a11yproject.com/checklist/) — Lista interativa baseada nas WCAG com explicações e exemplos práticos.  
- [**Deque University**](https://dequeuniversity.com/) — Plataforma educacional com cursos e materiais sobre acessibilidade digital.
